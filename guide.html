Giveaway RouletteTwitch · v1.1

A live giveaway picker for Twitch streams. Viewers type !join in chat, and a sideways-scrolling roulette picks the winner — with rewards, images, weighted odds, a countdown, and auto chat announcements. Nothing to install: it runs as OBS browser sources.

Add it to OBS

Two pieces from the same page — the wheel goes on your stream, the controls stay private in a dock.

On stream https://abdulazizjhk.github.io/giveaway-wheel/wheel.html?view=stage
Your controls https://abdulazizjhk.github.io/giveaway-wheel/wheel.html?view=dock
  1. In your scene, Sources → ➕ Browser → paste the On stream URL → set 1920 × 1080 → uncheck “Shutdown source when not visible”. It’s transparent, so it sits over your scene.
  2. Top menu Docks → Custom Browser Docks… → name it “Giveaway” → paste the Your controls URL → Apply. Dock it wherever you like — only you see this.
Seeing an old version after an update? Grab a fresh On stream URL with the 🔗 Copy source URL button in your dock — it now stamps the current version (e.g. &v=1.1) so pasting it forces a clean load. You can also just right-click the source → Refresh, or manually bump the &v= number on the URL.

Run a giveaway

  1. In the dock, type your Twitch channel → Connect (no login needed — it reads chat anonymously).
  2. Set a Timer (e.g. 60s) and click Open Entries. Viewers type !join to enter — the on-stream banner shows the live count + countdown.
  3. When the timer ends it auto-spins (the roulette scrolls and lands on a winner). Or press Spin / Space any time.
  4. The winner is revealed with confetti — and if you added rewards, the prize (and its image) shows too.

Rewards & prizes

  • Add prizes in the Rewards list. In Entrants mode the winner automatically gets the top reward; in Rewards mode the roulette picks the prize and gives it to a random entrant. Either way, one clean result.
  • Click the 🖼️ next to a reward to attach an image (logo / prize art) — it reveals big when that prize is won.

Everything you can do

!join entries — auto-collected from chat
Weighted odds — repeat !join = more chances
Sub / VIP / Mod only — gate by chat badge
Countdown — with 3-2-1 beeps
Auto-spin when the timer ends
Hide until spin — reveal on cue
Reward images — shown on the win
Chat announcements — @winner won {reward}
Banner position — place it anywhere
Colour glow that pulses on the win

Chat announcements optional

Want the bot to post “🏆 @winner won {reward}!” to chat? In the dock’s Chat announcements section:

  1. Get an OAuth token at twitchtokengenerator.com (scopes: chat:edit, chat:read).
  2. Enter your channel/bot username + the token → Connect bot → tick Post event messages. The token stays only in your browser.
